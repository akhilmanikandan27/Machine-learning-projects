# Machine Learning Projects

A collection of Machine Learning and Data Analysis projects developed using Python, Scikit-learn, Pandas, and Jupyter Notebook.  
This repository contains multiple projects focused on predictive modeling, exploratory data analysis, and practical machine learning applications.

---

## 📌 Repository Overview

This repository showcases various Machine Learning projects created for learning, practice, and improving data science skills.

The projects include:
- Prediction Models
- Exploratory Data Analysis (EDA)
- Regression Projects
- Classification Projects
- Data Visualization
- Real-world Dataset Analysis

---

## 🛠️ Technologies Used

- Python 3
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 📂 Projects Included

### 📈 Sales Prediction using Advertising Dataset
Predicts sales based on advertising expenditure using Linear Regression.

### 📊 Stock Price Prediction
Analyzes historical stock market data and predicts stock prices using machine learning techniques.

### ❤️ Heart Disease Prediction
Predicts the likelihood of heart disease using medical and health-related datasets.

### 💔 Divorce Prediction
Uses classification algorithms to predict divorce outcomes based on relationship-related attributes.

### 🚢 Titanic Data Exploration
Performs Exploratory Data Analysis (EDA) on the Titanic dataset to identify survival patterns and insights.

## 🚀 How to Run

1. Clone the repository

```text
git clone https://github.com/your-username/machine-learning-projects.git
```

2. Open the project folder

3. Install required libraries

```text
pip install pandas numpy matplotlib seaborn scikit-learn
```

4. Run the Jupyter Notebook

```text
jupyter notebook
```
